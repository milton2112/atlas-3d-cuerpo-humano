Atlas educativo interactivo

Atlas 3D del Cuerpo Humano

Un recorrido visual para entender como trabajan juntos los sistemas del cuerpo humano, pensado para alumnos que exploran solos o para proyectar en clase.

1. Elegi un sistema 2. Mira que organos lo forman 3. Abri puntos interactivos 4. Recorre como presentacion

Para alumnos

Elegir un sistema, observar el modelo y abrir organos clave. La ficha resume lo importante sin textos largos.

Para docentes

Usar pantalla completa y modo clase. Cada paso tiene una idea central, organos clave y una pregunta para conversar.

Si un modelo falla

La app muestra una vista temporal para que la clase continue.
