Atlas educativo interactivo

Atlas 3D del Cuerpo Humano

Un atlas visual para explorar organos, procesos y relaciones del cuerpo humano con un recorrido claro, moderno y facil de compartir por link.

1. Entra a un sistema 2. Observa el modelo y los organos clave 3. Abre fichas y puntos interactivos 4. Recorre el proceso completo

Como empezar

Entra a un sistema, mira el modelo principal y abre sus organos clave. La ficha ya resume lo mas importante para una lectura rapida.

Como recorrer

Usa pantalla completa y el recorrido guiado para avanzar paso a paso. Cada etapa deja una idea central y un punto concreto para observar.

Si un modelo tarda

La app muestra una vista temporal o un mensaje claro para que la exploracion pueda continuar sin cortar el recorrido.
